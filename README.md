## 2. 矩阵的压缩存储

### 2.1 理论依据

1. 利用三元组来存储稀疏矩阵
2. 利用三元组顺序存储稀疏矩阵时，要满足行主序排列
3. 算法思想为预先计算出每一列的第一个元素存储的位置，然后利用之前的结果可以快速的定位到每一个元素应该在的位置
